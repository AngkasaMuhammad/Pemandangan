# Pemandangan
Petarunx menghajar monyet agar tidak mendekat ke pohon agar dunia tetap cerah.  
  
### Desktop  
Lari = WASD  
Serang = mouse down  
  
### HP
Serang = long touch  
Lari = long touch sambil digeser
